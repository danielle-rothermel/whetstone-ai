What it proposes

Execute a seeded abstract rewrite system to its unique normal form

The generator emits an instance made of 3–6 productions over invented symbols plus a start string of 8–14 symbols. The system is constrained to be confluent and terminating (or a fixed leftmost-innermost strategy is imposed) so the normal form is unique and program-checkable. The model applies the productions, iterating to fixpoint (not a single pass), and returns the short normal-form string — or, in the L-system parallel-rewriting stratum, a bounded derived feature (final length, or the symbol at position k) so the answer stays short even though a D0L derivation blows up exponentially. Scoring is exact match against a reference rewriter run to fixpoint. Latent rules decompose cleanly: each production, redex-selection strategy, iterate-to-fixpoint, empty-production deletion, guard conditions, and parallel-vs-sequential application — so failures localize to a single un-applied production or a halted iteration.

Academic anchor & lineage

Literature parents

  • Terese, Term Rewriting Systems (eds. Bezem, Klop & de Vrijer, CUP 2003) — foundational reference for abstract rewriting, confluence, and termination underpinning the normal-form oracle. The merged file attributed this vaguely to "Terese; confluence-strategy work 2024"; corrected here — "Terese" is a collective pseudonym for editors Bezem, Klop & de Vrijer.
  • Baader & Nipkow, Term Rewriting and All That (CUP 1998) — standard textbook; source for Newman's lemma (local confluence ⇒ confluence for terminating systems) guaranteeing a unique normal form. Cited only in the research file, not the merged list; a core reference that belongs here.
  • Zhang, Mitterwallner et al., Automated Strategy Invention for Confluence of Term Rewrite Systems (arXiv:2411.06409, 2024) — the "confluence-strategy work 2024" the merged file gestured at; supports enforcing unique normal forms.
  • Prusinkiewicz & Lindenmayer, The Algorithmic Beauty of Plants (Springer 1990) — canonical L-system reference; grounds the parallel-rewriting (D0L) stratum.
  • Lindenmayer, Mathematical models for cellular interactions in development (J. Theoretical Biology, 1968) — canonical original L-system paper; not fetched to a confirming page this session.

Supports

Against
  • Scored at the 26-tier with tie-breaker one, the flag being that it is potentially too hard — the difficulty may sit above the band where an incremental ladder is observable

Optimization-path skeptic

Supports

Against

Against

Implementation: steps & risks

Implementation scoring was produced by the same pipeline as every other candidate — web discovery + desk scoring with the same rubric and exercises + a deep active-path code read written to repos/ — and folded into the official ranking by the 2026-07-21 amendment. Run-verification: open-thought/reasoning-gym was run-verified during round 2 (under c24: same-seed byte-identical, 6/6 independent ground-truth checks), so repo-level verification carries over; the c04-specific rewrite-system logic is authored on top of that reused, verified infrastructure.

Adopt reasoning-gym as substrate

Reuse ProceduralDataset, per-instance Random(seed+idx) seeding, register_dataset, and the exact-match score_answer (already strict 0/1)

Clone-and-generalize ab.py (Option B, ~150 LOC)

Sample a nonce alphabet, a rule table, and a start term from the seed; generalize compute_steps from four hard-coded productions to a rule-table lookup; render the prompt from the sampled table

Enforce determinism before freeze

Certify confluence + termination (critical-pair check / Newman's lemma) or impose a fixed leftmost-innermost strategy so every instance has a unique normal form

Add latent-rule and L-system strata

Copy the curriculum pattern for rule-count / alphabet-size levels; add a capped-iteration D0L stratum asking for a bounded derived feature; add the optional MC-over-candidate-normal-forms variant

Calibrate difficulty and pin the noise floor

Tune term length and rule count so a default prompt lands mid-range; measure temp-0 character-level noise on longer normal forms and, if needed, shorten outputs or move to the bounded-feature / MC framing

Risks

  • Add-one-rule blast radius is global — a new production can break confluence/termination via critical-pair overlaps, so the confluence check must re-run on the whole system and every instance whose normal form changes must be re-verified and regenerated; not rule-local.
  • Four-plus pre-freeze gates — determinism enforcement, "potentially too hard" difficulty calibration, the char-arithmetic temp-0 noise floor on longer normal forms, and the L-system iteration cap must all clear before freeze.
  • Two open repair flags, no skeptic pass — landscape.md records "potentially too hard" and "char-arithmetic temp-0 noise"; both repairable but non-empty, and no adversarial lens examined c04 below the 28-tier.
