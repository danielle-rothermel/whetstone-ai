GATED. This candidate scores well on structure but was refuted on the optimization path outright, and the round-2 gate is categorical: a candidate whose measurement story was refuted in round 1 without a concrete rescue cannot be recommended no matter how easy it is to build. The refutation is specific and hard to dodge. The per-instance gloss table hands the latent morphology directly to the model in context, so there is no hidden knowledge left for a better prompt to discover; and the one rule meant to carry the difficulty — that a case suffix determines subject versus object regardless of linear position — is the highest-prior rule in all of linguistics, so a single competent paragraph one-shots most of the gap. The only fix on offer, an MTOB-style redesign that withholds affix semantics from the gloss, changes what the task is, which makes it a different candidate, not a rescue of this one. The implementation total is low and academic here: even a perfect build cannot lift a gated candidate off the floor, and the one repo that touched this cluster offers essentially nothing reusable for morphology.

What it proposes

Learn a per-run agglutinative micro-grammar in context, then translate one phrase to a rigid template

A generator builds a fresh agglutinative micro-language on every run: random roots plus affixes that mark case, number, and tense, together with a topic-fronting particle. One instance is a single conlang sentence accompanied by a tiny gloss table; the model must return an English paraphrase in a rigid three-to-six-word template with fixed articles and fixed slot order, scored by exact match after trivial normalization. The latent rules are meant to layer: a case suffix fixes subject/object independent of position (the load-bearing rule naive SVO prompts miss), number and tense affixes, article insertion, a fronting reorder, and an inflection stratum with allomorphy. Ground truth is a deterministic template filler; randomized morphology per run is contamination-proof, and MTOB/LingOly document large headroom for grammar-from-context learning.

Example instance

Illustrative — reconstructed faithfully from the merged summary; the corpus gives no canonical instance for this gated candidate.

Gloss table:
  roots:   tavu = dog,  miron = cat,  kela = chase
  affixes: -ok = subject (nominative),  -ip = object (accusative)
           -sa = plural,  za = (topic-fronting particle, sentence-initial)

Conlang sentence:   za mironip tavuok kela
Rigid EN template:  [art] [subj] [verb] [art] [obj]
Gold answer:        the dog chases the cat

Naive-SVO miss:     "the cat chases the dog"
  — a word-order prompt reads the fronted "mironip" (topic particle za +
    object suffix -ip) as the subject; the case suffix, given right in the
    gloss, is what the model was supposed to apply.

The miss shows the whole problem: the rule that produces the error is spelled out in the gloss table the model is handed, so recovering it is comprehension, not induction.

Implementation: steps & risks

Build the morphology generator from scratch

No reusable code exists; write a seeded generator that samples roots plus a rotating inventory of case/number/tense affixes, a topic-fronting particle, and an allomorphy stratum, emitting a conlang sentence plus a tiny gloss table per instance.

Write the deterministic template filler

Implement the oracle that maps a parsed sentence to the rigid three-to-six-word English template (fixed articles, fixed slot order) and canonicalize for exact match.

Attempt the gate rescue before anything ships

The task is ineligible as designed; the only path to eligibility is the MTOB-style redesign that withholds affix semantics from the gloss so morphology must be induced — but that is a different candidate and must be scoped and re-scored as one, not slipped in as a patch.

Risks

  • One-shot collapse (the fatal one): the gloss table hands over the morphology and case-beats-word-order is the highest-prior linguistic rule, so a single competent paragraph closes most of the gap and the incremental ladder disappears.
  • Rescue changes the task: withholding affix semantics to restore headroom converts this into an induction task (a different candidate), so "fixing" it forfeits its identity rather than salvaging its score.
  • From-scratch build cost with no payoff: the morphology, glossing, and allomorphy stratum must all be written new, and even a flawless build cannot lift a gated candidate off the floor.
